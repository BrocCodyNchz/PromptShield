PromptShield Privacy Policy

Last updated: February 2026

Overview

PromptShield is a browser extension that helps protect you from accidentally sending sensitive data (such as API keys, passwords, or credit card numbers) to AI chatbot websites. This policy explains how we handle data.

Data We Do Not Collect

PromptShield does not collect, transmit, or store any of the following:

  • Text you type or paste into AI chat websites
  • Personal information (name, email, address, etc.)
  • Browsing history or website URLs
  • Analytics or usage data

All scanning happens in your browser. No data is sent to any server.

Data Stored Locally

PromptShield stores only the following data on your device:

  • Enabled state — Whether the extension is turned on or off
  • Session warning counts — How many warnings were shown in the current session, broken down by category (e.g., "3 API keys"). These are category names and numbers only; no actual sensitive data is stored.

This data stays on your device and is not transmitted anywhere.

Third-Party Services

PromptShield does not use third-party analytics, advertising, or tracking.

Your Rights

Because we do not collect personal data, there is no personal data to access, correct, or delete. You can uninstall the extension at any time to remove all locally stored data.

Changes

If we change this policy, we will update the "Last updated" date above. If the extension is distributed via the Chrome Web Store, we will notify users of material changes.

Contact

For questions about this policy, open an issue on the PromptShield GitHub repository or contact the developer.
